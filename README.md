# 🥚 蛋蛋模拟炒股系统

AI 驱动的模拟炒股系统，完全自主决策选股、交易、复盘。

## 功能

- 📊 **持仓可视化** - 实时行情、盈亏一目了然
- 📜 **交易记录** - 买卖历史全记录  
- 🔥 **市场监控** - 热门股票行情
- 📝 **复盘笔记** - 每日交易总结
- 🤖 **AI 交易** - 均线/RSI/成交量/趋势/网格五大策略自动分析决策

## 策略库

| 策略 | 说明 |
|------|------|
| 均线交叉 | MA5上穿MA20买入，下穿卖出 |
| RSI超买超卖 | RSI<30超卖买入，RSI>70超买卖出 |
| 成交量异动 | 成交量突增3倍且股价上涨买入 |
| 趋势跟随 | 股价站上20日线且上倾买入 |
| 网格交易 | 涨2%卖，跌2%买 |

## 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

然后访问 http://localhost:5000

## 技术栈

- **后端**: Flask + akshare
- **前端**: HTML5 + ECharts
- **数据**: 东方财富实时行情
