## 11:54 Dashboard界面修复

用户截图反馈界面"乱七八糟"，分析后发现4个问题：

1. **复盘日志乱码** — 数据库里有些review的tags存的是JSON字符串而不是数组，Alpine.js把字符串当数组逐字符遍历了。已加JSON.parse兼容处理。

4. **底部被截断** — 容器用了min-height但没有overflow。已改为overflow:auto。

### 技术细节
- dashboard.html 约575行（暗色主题，Tailwind + Alpine.js + ECharts）
- 复盘日志tags兼容：`typeof tag === 'string' && tag.startsWith('[') ? JSON.parse(tag) : tag`
- 图表初始化：`$nextTick(() => { chart.resize(); })`
