## ⚠️ A股交易规则（必须遵守！）

### 交易时间
- 集合竞价：9:15-9:25（可挂单，不可成交）
- 开盘竞价：9:25-9:30
- **连续竞价：9:30-11:30 / 13:00-15:00**
- 收盘定价：14:57-15:00

### 交易单位
- 最少**100股（1手）**
- 每次买卖必须是100股的整数倍

### 涨跌幅限制
- 普通股票：±10%
- ST股票：±5%

### 手续费（实际）
| 类型 | 费率 | 收取时机 |
|------|------|----------|
| 印花税 | 0.05% | **仅卖出时** |
| 过户费 | 0.001% | 仅沪市 |
| 券商佣金 | ~0.03% | 买卖均收 |

### T+1制度
- **当日买入，下一交易日才能卖出**

## ⚕️ 交易前必读：查阅历史复盘

**每次下单前，必须先查阅历史复盘记录！**

1. 调用 `/api/daily` 获取历史复盘
2. 回顾最近5次交易记录 `/api/trades`
3. 检查是否有重复失误、是否有可复制的成功模式
4. 根据历史调整当日策略

**进化机制：**
- 每次复盘记录盈亏和操作理由
- 分析成功/失败原因
- 优化策略参数（RSI阈值、止盈点等）
- 避免重复同样的错误

## 一、买入规则

### 买入时间窗口
- **14:30 - 14:55**（尾盘买入）
- 14:55后不再下单，避免尾盘异动

### 个股条件（全部满足）

| 条件 | 要求 |
|------|------|
| 涨幅 | **3% - 5%** |
| 成交量 | **> 1.5倍** 5日均量 |
| 换手率 | **3% - 10%** |
| 流通市值 | **50亿 - 200亿** |
| RSI | **40 - 65**（不超买不超卖） |
| 分时均价线 | 当前价格 **站上** 分时均价线 |
| 强于大盘 | 个股涨幅 **>** 大盘涨幅 |

---

## 二、卖出规则（次日早盘冲高即走）

### 核心原则
- **一夜持股，次日了结**
- 不贪不恋，冲高就跑

### 卖出时机
- **次日09:30 - 10:30** 冲高卖出
- 若开盘直接低开，观察15分钟内是否有反弹，无反弹立即止损

### 卖出条件（满足任一即卖出）

| 条件 | 操作 |
|------|------|
| 次日冲高 **+3%~5%** | 止盈卖出 |
| 次日冲高回落超 **-2%**（从最高点） | 立即卖出 |
| RSI > 70 | 卖出 |
| 放量滞涨（涨幅<1%，成交量暴增） | 卖出 |
| 开盘跌破分时均价线且收不回 | 卖出 |

---

## 三、止损规则

- 个股亏损达到 **-3%** → 立即止损（无条件）

---

## 四、仓位管理原则

- **宁错过，不做错** — 不符合规则坚决不动
- 仓位自由决定，不设硬性上限

---

## 五、复盘记录保存

**每次复盘必须包含：**
- 今日账户盈亏（总市值 - 初始资金）
- 持仓情况（每只股票的现价、成本、盈亏%）
- 交易记录（买入/卖出的股票、时间、价格、理由）
- 操作反思（这次操作对在哪/错在哪）
- 明日策略调整（根据今天学到的调整）

**调用API保存：**
```bash
curl -X POST http://localhost/api/review \
  -H "Content-Type: application/json" \
  -d '{"content": "复盘内容", "tags": ["标签"], "strategies": ["策略"]}'
```

**不复盘=没记录！必须写进系统！**
